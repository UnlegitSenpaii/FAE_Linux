# Security Policy

## Supported Versions
Only the most recent version in the release tab is supported.<br>
Found here: https://github.com/UnlegitSenpaii/FAE_Linux/releases

## Reporting a Vulnerability

**Low severity / non-sensitive issues:** Open an issue on the [issues page](https://github.com/UnlegitSenpaii/FAE_Linux/issues). Please include as much detail as possible about the vulnerability and how to reproduce it.

**Urgent or sensitive issues:** For private disclosure, send a Discord DM to `senpaii.dev`. I will try to respond in the evening (CET) on weekdays, or over the weekend.

Please avoid publicly disclosing details of a sensitive vulnerability before it has been addressed.
